-------USE INTRUCTIONS------------

1) install python3 in your os (python3.7 , 3.8 or 3.9)

2) cd app directory

3) activate venv  :

    source venv/bin/activate

4) install dependecies :
    pip install -r requirements.txt

5) start the app :

    export FLASK_APP=App 
    flask run
